About UniEvent

A cloud-native university events platform built on AWS

What is UniEvent?

UniEvent is a university events portal developed as part of the CE408 Cloud Computing course at Ghulam Ishaq Khan Institute (GIKI). It connects students and staff with live event listings sourced from the Ticketmaster Discovery API, covering music, sports, and arts events across multiple regions.


The application is built with Flask and served by Gunicorn on Amazon EC2. An APScheduler background job fetches the latest 20 events from the Ticketmaster API every 15 minutes and stores them in memory, ensuring the page always reflects up-to-date listings without requiring a database. All infrastructure runs on AWS in the eu-north-1 (Stockholm) region, with multi-AZ deployment for high availability.

Platform at a Glance

  • Events per refresh 50
  • Refresh interval 15 minutes
  • Data source Ticketmaster API
  • AWS Region eu-north-1 (Stockholm)
  • Availability Zones 2 (multi-AZ)
  • Health check /health
AWS Architecture
Component Service Purpose
Load Balancer Application Load Balancer (ALB) Single internet-facing entry point; distributes traffic and performs health checks
App Servers (×2) Amazon EC2 (t2.micro) Runs Flask + Gunicorn across two Availability Zones for fault tolerance
Virtual Network Amazon VPC Isolates all resources; public subnets for ALB, private subnets for EC2
Internet Access Internet Gateway + NAT Gateway IGW for inbound traffic to ALB; NAT Gateway for EC2 outbound API calls
Static Storage Amazon S3 Hosts the GIKI logo and any static media assets
Identity IAM Role + Inline Policy Grants EC2 least-privilege read/write access to the S3 bucket only
External Data Ticketmaster Discovery API Provides live event listings fetched every 15 minutes by the scheduler
